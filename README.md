# 🌍 智能旅行规划助手

一款智能旅游攻略生成工具。输入出发地、目的地、出行方式和偏好，自动搜索景点信息并生成每日行程方案。

---

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🏙️ 城市选择 | 出发地/目的地自动补全，支持热门城市快捷选择 |
| 🚀 出行方式 | 飞机 / 火车 / 自驾，自动计算路线和费用 |
| 📅 日期规划 | 选择出行日期，自动计算天数，最多支持14天 |
| 🎯 偏好选择 | 自然风光/历史人文/美食/购物/亲子等7种类型多选 |
| 🔍 联网搜索 | 调用高德API获取景点数据，Web搜索获取实时信息 |
| 🕐 实时信息 | 景点开放时间、门票价格、最佳观赏季节 |
| 📋 智能排程 | 基于评分+聚类+路径优化的行程自动生成 |

## 🚀 快速开始

### 前置条件

- [Node.js](https://nodejs.org/) >= 16
- 注册 [高德开放平台](https://lbs.amap.com/) 账号，获取 **Web服务 API Key**

### 安装 & 运行

```bash
# 安装依赖
npm install

# 配置高德 API Key（二选一）
# 方式1：环境变量
set AMAP_KEY=你的高德Web服务Key    # Windows
export AMAP_KEY=你的高德Web服务Key # macOS/Linux

# 方式2：直接修改 server.js 中第32行的 AMAP_KEY

# 启动服务
node server.js
```

浏览器打开 `http://localhost:3000` 即可使用。

## 📊 数据流

```
用户输入表单 → 校验通过
    ↓
并行 → 搜索景点 + 计算路线
    ↓
生成 → 评分→聚类→排天→路径优化→信息增强
    ↓
行程展示（按天时间线）
```

---

## 🔧 核心算法（5步流水线）

| 步骤 | 说明 |
|------|------|
| 1. 评分 | 5个维度加权评分：类型匹配(30%) + 评分(20%) + 热度(20%) + 季节(20%) + 数据完整度(10%) |
| 2. 聚类 | 基于Haversine距离，3km半径内景点归为一组 |
| 3. 排天 | 每天3-5个景点，同聚类尽量同天，上午/午餐/下午/晚间时段 |
| 4. 优化 | 最近邻贪心算法优化当天游览顺序，减少路程 |
| 5. 增强 | 将Web搜索结果中的开放时间/门票/季节合并到景点数据 |

### API 调用策略

- **缓存策略**：POI搜索1小时、路线6小时、城市建议30分钟
- **兜底策略**：API失败返回空结果（不阻塞流程）
